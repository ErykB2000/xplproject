xPL App Title: xPLClock

Default Source.Instance:
TONYT-CLOCK.<hostname>HHMMSS

Default Filter(s):
xpl-cmnd.*.*.*.datetime.*

Default Group(s):
none

xPL Schema(s):
DATETIME.BASIC


Purpose:
To provide date/time functions to a xPL systems


Installation:
Requires xPL OCX
Accepts Standard xPL Settings
Additional Remote Config Items:
CLOCK = seconds 1 to 60 for each update, default = 60
FORMAT1 = user defined date/time format, default = ""
FORMAT2 = user defined date/time format, default = ""
FORMAT3 = user defined date/time format, default = ""
FORMAT4 = user defined date/time format, default = ""
