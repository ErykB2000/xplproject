xPL App Title: xPLDGEnd, xPLDGStart, xPLDGRemind, xPLDGPlaying

Default Source.Instance:
TONYT-DGUIDE.GUIDE

Default Filter(s):
NONE

Sends xPL Message(s):
Message Type=XPL-TRIG (or XPL-STAT for Playing)
Class=DGUIDE
Type=END, START, REMIND, PLAYING


Purpose:
To provide Digiguide programme notification to xPL systems

Notes:
xPLDGRemind REQUIRES Digiguide 6.0 or better

Installation:
Copy 
xPLDGEnd.dgaddin
xPLDGStart.dgaddin
xPLDGRemind.dgaddin
xPLDGPlaying.dgaddin
to Digiguide folder

Remote Config Items:
NONE
