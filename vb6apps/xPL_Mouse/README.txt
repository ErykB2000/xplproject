xPL App Title: xPLMouse

Default Source.Instance:
TONYT-MOUSE.<hostname>HHMMSS

Default Filter(s):
none

Default Group(s):
none

Schema(s):
REMOTE.BASIC


Purpose:
Provide xPL remote control messages from a Marmitek RF mouse and receiver
Also allows mouse control on attached PC


Installation:
Requires xPL OCX
Requires Marmitek Mouse and Receiver
Accepts Standard xPL Settings
Additional Remote Config Items:
COMPORT=
BAUD=
DATABITS=
PARITY=
STOPBITS=
FLOWCONTROL=
RTSENABLE=
DTRENABLE=


Notes:
Only COMPORT should need configuring
