xPL App Title: xPLTTS

Default Source.Instance:
TONYT-TTS.<hostname>HHMMSS

Default Filter(s):
XPL-CMND.*.*.*.TTS.*

Default Group(s):
none

Schema(s):
TTS.BASIC

Purpose:
To speak text via TTS


Installation:
Requires xPL OCX
Requires MS TTS Engine (or any SAPI compliant?)
Accepts Standard xPL Settings
Additional Remote Config Items:
VOICE=<sapi voice name, or part name>
VOLUME=<0 to 100>
SPEED=<-10 to 10>


Notes:
Default Voice is first enumerated from sapi
Default Volume is 100
Default Speed is 0
