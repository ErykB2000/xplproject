xPL App Title: xPLWM918

Default Source.Instance:
TONYT-WM918.<hostname>HHMMSS

Default Filter(s):
none

Default Group(s):
<XPL-GROUP.<group>> or none

Schema(s):
WEATHER.STATUS
WEATHER.CURRENT (every 60 seconds and on demand)
WEATHER.LO (on demand)
WEATHER.HI (on demand)
WEATHER.RAIN (on demand)
WEATHER.ALARM (on alarm)


Purpose:
To provide weather information from a Weather Station


Installation:
Requires xPL OCX
Accepts Standard xPL Settings
Additional Remote Config Items:
COMPORT=
FREEWXIP=


Notes:
If using FreeWX with your Weather Station set it to broadcast to the IP address of the PC that this PC is running on
Set FREEWXIP to the IP address of the PC that FreeWX is running on (can be same PC)
