System Language Extensions

There are basically two objects specific to xPLHal.

$Sys Class

The sys class provides extensions to PowerShell which allow the core elements of the xPLHal server to be manipulated. All of the system classes are referenced by the object name "$Sys". Because PowerShell is case insensitive, $sys is also valid.

$Msg Class

This class gives access to the xPL message the script reacted on, if there was one. Otherwise, the object contains a null value. This will be the case with manual script invocation.
