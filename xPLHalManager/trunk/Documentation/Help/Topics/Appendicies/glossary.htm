Glossary

  • Construct
    An item of data that forms part of a schema used to determine which sub-routines to execute when an incoming message is received. Examples of constructs include an xPL message class, an X10 house code, the period, or the mode.
  • Function
    A block of code which performs a specific task and returns a value to the caller.
  • Script
    A file containing one or more functions/sub-routines.
  • Sub-routine
    A block of code that performs a specific task and does not return a value.
  • X10
    A powerline carrier technology that allows lights and appliances to be switched on from anywhere throughout the home by sending control signals over the existing mains wiring.
