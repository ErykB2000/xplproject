Importing X10 Devices from HomeVision

This feature allows you to import your X10 device names from HomeVision, saving you the job of entering all your X10 device information into xPLHal by hand. It will only import devices that have been renamed, e.g. not named "X10 A1" and so on.

To import your devices, follow the steps below:

In the HomeVision PC client:
  • Make sure the latest schedule is loaded into the application.
  • Goto File-Export Schedule.
    This will save a file with yourschedulename.hvx to the HomeVision program directory (normally c:\Program Files\HomeVision\ but sometimes it can save the file to c:\)
In xPLHal Manager:
  • Select Tools -> Import HomeVision X10 Devices.
  • When the Open File dialog box appears, select the HomeVision schedule file that you saved previously.
  • You will receive a message telling you how many devices were imported.
  • Select (or refresh) the X10 device list in xPLHal Manager to see your newly imported devices.
