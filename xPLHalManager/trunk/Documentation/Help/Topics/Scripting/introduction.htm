Introduction to Scripting in xPLHal

The core of xPLHal is it's ability to execute VBScript routines in response to received xPL messages or timed events.
It is recommended that you familiarise yourself with the VBScript language, as well as the section on xPLHal Concepts.
This section is intended to provide you with an overview of the xPLHal scripting system - it is not a guide to the VBScript programming language. Please refer to the MSDN for full documentation on the VBScript language.

Modifying and Reloading Scripts

 It is important to realise that any changes made to scripts will not take effect until either the xPLHal service is restarted, or the Reload Scripts option is selected from the Scripts context menu in the xPLHal Manager.
This allows you to modify a number of scripts, and have the changes take effect only when you have finished modifying all your scripts.

Language Extensions

In addition to the standard functions of the VBScript language, scripts that are executed by the xPLHal Scripting Engine have access to a number of extra classes, designed to assist you in getting the most out of your xPLHal Home Automation System.
