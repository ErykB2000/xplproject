Monitor

The monitor is a tool that allows you to view xPL messages on your network.

It provides a list of xPL messages, specifying their type, source, target, schema, time and hop. Details of the contents are provided at the bottom of the page when a message is highlighted in the list.

The filter provides a method of limiting the number of messages listed. If the filter is enabled, then only messages of source, schema or type that are ticked will appear in the list. Any messages with a source, schema or type that is not listed will not appear in the list. If the filter is not enabled, all messages will be listed.
Note that if you tick only one item on one of the tabs and leave all other items unticked, you will see NO xPL messages. You must have at least one item ticked on each tab before you will see any xPL messages. The simplest way to use the filter is to tick all the items then untick the items on a single tab that you do not wish to see messages for.

  • Refresh: This will re-populate all the items in the filter options.
  • All: This will select all the filter options.
  • None: This will unselect all the filter options.
  • Start: This will start the monitor. The monitor is started automatically when the monitor is loaded.
  • Stop: This will stop the monitor.
  • Clear: This will clear the list of logged messages.
  • Stats: This will list some information about the received messages.
  • Save: This will save the list of logged messages in a .csv file.
  • Exit: This will close the Monitor.
